# Change Log

All notable changes to the "UI Insight" extension will be documented in this file.

## [1.0.0] - 2025-12-25

### Added
- Initial release
- AI-powered component analysis using OpenAI, Anthropic, or Groq
- AST-based code parsing with Babel
- Pattern detection for layout and accessibility issues
- Automatic code refactoring with JSX transformation
- Support for Tailwind CSS, Styled Components, and CSS Modules
- Webview panel for displaying analysis results
- Context menu integration for React files
- Configuration options for AI provider and styling preferences

### Features
- Detects excessive nesting and suggests component extraction
- Identifies accessibility issues (missing alt, ARIA labels)
- Recognizes repeated patterns suitable for map() operations
- Suggests semantic HTML replacements for div-heavy structures
- Provides consistent spacing recommendations
- One-click code refactoring with preview
